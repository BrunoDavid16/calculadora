# 🧮 Calculadora Python (Tkinter)

Este projeto consiste em uma calculadora funcional com interface gráfica, desenvolvida em Python utilizando a biblioteca Tkinter. O objetivo foi praticar lógica de programação, manipulação de eventos e design de interfaces (GUI) aplicados a ferramentas de uso diário.

## 🚀 Funcionalidades
- **Operações Básicas:** Soma, subtração, multiplicação e divisão.
- **Interface Moderna:** Layout com cores escuras (Dark Mode) e botões de destaque.
- **Limpeza de Visor:** Botão "C" para resetar os cálculos rapidamente.
- **Lógica de Precisão:** Tratamento de números decimais e operações sequenciais.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Biblioteca GUI:** Tkinter (Nativa do Python, garantindo leveza e compatibilidade).

## 📂 Como Rodar o Projeto Localmente

1. Certifique-se de ter o Python instalado em sua máquina.
2. Clone este repositório:
   `git clone https://github.com/BrunoDavid16/calculadora.git`
3. Acesse a pasta do projeto pelo terminal:
   `cd calculadora`
4. Execute o script:
   `python calculadora.py`
